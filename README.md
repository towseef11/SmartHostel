# 🏠 SmartHostel - Hostel Management System

SmartHostel is a full-stack web application designed to digitize and simplify hostel operations. It provides separate dashboards for users and owners to manage 
daily hostel activities efficiently, eliminating manual work and improving transparency.


## ✨ Why SmartHostel?

Traditional hostel management is slow, manual, and error-prone. SmartHostel solves this by bringing everything into a single digital platform — from 
room allocation to complaints handling and food menu management.


## ⚙️ Tech Stack

**Frontend:**
- React.js
- HTML, CSS, JavaScript

**Backend:**
- Java (Spring Boot)
- Spring Security + JWT Authentication
- REST APIs

**Database:**
- MySQL



## 🔑 Key Features

### 👤 User Side
- Secure login system (JWT based)
- View assigned room details
- Book washing machine time slots
- Submit and track complaints
- View weekly food menu
- Personal dashboard

### 🏢 Owner Side
- Manage users and rooms
- View and resolve complaints
- Update food menu weekly
- Monitor hostel activity
- Centralized control dashboard


## 🧠 Core Concepts Implemented

- REST API design
- Authentication & Authorization (JWT)
- Role-based access control (User / Owner)
- CRUD operations
- React component-based architecture
- Spring Boot layered architecture (Controller → Service → Repository)
